# SPACE SHOOTER

## 制作時期

2025年12月～2026年1月

## 作品の説明

本作は、Pythonのゲームライブラリ「Pygame」を用いて開発された、王道の横スクロール型2Dシューティングゲームです。プレイヤーは宇宙戦闘機を操作し、迫りくる敵の群れを撃破しながら、宇宙、地表、敵基地へと変化するステージを攻略していきます。

ゲームプレイはシームレスに進行し、スコアや生存時間に応じて敵の攻撃パターンや背景が変化していくのが特徴です。序盤は宇宙空間での空中戦ですが、中盤からは山岳地帯などの「地形」が登場し、敵の攻撃だけでなく壁や地面への衝突も回避する必要があります。

登場する敵キャラクターの動きには多彩なアルゴリズムを実装しています。単純に突撃してくる敵だけでなく、三角関数を用いて波打つように飛来する戦闘機、プレイヤーの座標を計算して執拗に追尾してくる敵、そして地上を跳ね回りながら攻撃してくるロボットなどがプレイヤーの行く手を阻みます。特に敵の発射する弾は、移動ベクトルに合わせて画像が回転するように計算されており、視覚的なリアリティを持たせています。

ステージの最後には、複数の砲台と高い耐久力を持つ巨大ボス「ビッグ・コア」が待ち構えています。ボスの攻撃を回避しながら弱点を突き、撃破することでミッションクリアとなります。また、ハイスコアはファイルに自動保存されるため、過去の自分とスコアを競うリプレイ性も備えています。レトロなアーケードゲームの緊張感と爽快感をPythonで再現することを目指して制作しました。

## 操作方法

キーボードを使用して操作します。

- **矢印キー（↑↓←→）**: 自機の移動
- **SPACEキー**: ショット発射
- **ゲームオーバー時**: 何かキーを押すとタイトルへ戻ります

## 使用ライブラリ / 動作環境

- Python 3.x
- Pygame

## インストールと実行方法

必要なライブラリをインストールした後、`main.py` を実行してください。

```bash
pip install -r requirements.txt
python main.py
```
